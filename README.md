### Code

The "Code/" folder contains the Jupyter notebooks I have used to obtain my results. The python libraries used are given in "requirements.txt".

#### SimpleFigures.ipynb

This notebook contains figures that do not need preliminary computations. 

#### PhotonOrbits-Mass.ipynb

This notebook contains a study of null geodesics in General Relativity and in Entangled Relativity. The distances are normalized with respect to the mass of the considered black hole.

#### PhotonOrbits-Horizon.ipynb

This notebook contains the same analysis as in "PhotonOrbits-Mass.ipynb" except the distances are normalized with respect to the horizon radius of the black hole. 

#### MassiveDynamics.ipynb

This notebook contains a study of timelike geodesics in General Relativity and in Entangled Relativity.

#### PhenoCircular.ipynb

This notebook contains a statistical estimation of the mass and the charge of a black via some ER-generated mock data.
